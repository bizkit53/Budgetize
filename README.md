# Budgetize - the budget management app
It makes it easier to understand your spending habits by categorizing your transactions and showing them in <br>eye-catching charts, so you can better plan your future budget to be more appropriate for your financial situation.
## The technology which stands behind
The application is built with **_Dart_** language and **_Flutter_** framework. Data handling is based on a local **_Hive_** database. <br>The visual side of the project uses the **_charts_flutter, fl_chart_**  and **_pie_chart_**  libraries to represent data in graphical charts and the **_author's own concept_** to make the application look consistent and aesthetically pleasing.
